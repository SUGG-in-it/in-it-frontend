**코드리뷰를 요청하고, 클린코드, 리팩터링, 아키텍쳐등 코드에 관해 피드백을 받을 수 있는 웹 사이트 입니다.

코드리뷰 해주는 사이트가 없다는 사실을 알고 직접 개발하였습니다.**

├── public

│   ├── apis
│   ├── assets
│   ├── components
│   │   ├── common [공통으로 사용되는 컴포넌트]
