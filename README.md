## 🗂️ Directory structure


```bash
├── public
├── src
│   ├── __mocks__
│   ├── apis
│   ├── assets
│   ├── components
│   │   ├── common [공통으로 사용되는 컴포넌트]
│   │   ├── hoc
│   │   └── [routes] [재사용이 되지 않고, 각 페이지에서만 사용되는 컴포넌트]
│   ├── constants
│   ├── hooks
│   │   └── queries
│   ├── layouts
│   ├── pages
│   ├── store
│   ├── styles
│   ├── types
│   └── utils
└── README.md
```
